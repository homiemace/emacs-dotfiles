* Quick Reference

** Runes (Svelte 5)
- ~$state()~ - Declare reactive variable
- ~$derived()~ - Compute from other state (cached)
- ~$effect()~ - Run code when dependencies change (DOM manipulation)
- ~$props()~ - Receive component props
- ~$bindable()~ - Mark prop as two-way bindable

** Old vs New Syntax
- ~on:click~ = ~onclick~ (no colon)
- ~export let prop~ = ~let { prop } = $props()~
- ~<slot />~ = ~{@render children()}~
- ~$: derived~ = ~const derived = $derived(...)~

** DOM References
- ~bind:this={element}~ - Get DOM node reference
- ~$effect(() => { element?.focus() })~ - Use ref after mount
